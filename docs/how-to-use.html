Getting Started Navigating the Archive

The Detroit Badman Archive offers two primary tools for exploring Black masculine heroism in Detroit: an interactive geospatial map and a network visualization. Both tools are navigable by mouse, keyboard, or screen reader. This guide explains how to use each one.

Geospatial Tool Using the Map

Markers: Each badman figure is represented by a marker on the map. Select any marker to view biographical information, key events, and external links. A figures list below the map provides an alternative way to select figures without using the map directly.

Modality Indicators: Markers are distinguished by both color and shape. Detective figures use a pin marker; Political Revolutionary figures use a star marker. The legend on the map page identifies each modality by its marker shape.

Territory Polygons: Shaded areas represent the geographic territories where each figure operated. Select a marker to toggle its territory polygon on or off. Double-select a polygon to remove it.

Navigation: Use scroll or the +/- controls to zoom, and drag to pan. Keyboard users can navigate with arrow keys to pan and +/- keys to zoom.

Relationship Tool Using the Network

Nodes: Each node represents a badman figure. Node size reflects the figure's influence score at the selected year, and node shape corresponds to the figure's modality. The legend identifies each modality by its shape.

Edges: Lines between nodes represent documented relationships—creator to creation, ideological alignment, geographic convergence, or organizational ties. Edge types are distinguished by both color and dash pattern.

Timeline Slider: Use the timeline slider at the bottom to filter figures by their period of activity. The current year is displayed beside the slider and updates as you adjust it.

Interaction: Select any node to highlight its connections and view detailed relationship information in the side panel. A figures list below the network provides an alternative way to select figures. You can pause or resume the network animation using the control button above the graph.

Understanding Entries Reading Profiles

Designation: Each figure is marked as Real, Fictional, or Meta-Badman (figures who both lived the badman life and created badman narratives).

Badman Score: Figures are evaluated on five criteria, each scored 1 to 5: Outlaw Relationship to Law, Community Authorization, Violence as Language, Cultural Preservation Function, and Hypermasculine Performance. The total score ranges from 5 to 25.

Sources: Every claim in a profile is linked to primary or secondary sources. Source links open in a new tab and are labeled accordingly.

External Links: Profiles include links to related content—films, books, speeches, and archival materials. All external links are marked when they open outside the archive.
